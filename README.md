# Portscanner
It's a available port scanner from a vulnerable website.

### Screenshot
<img src ="portscanner.png">

### Prerequisites

* git

* python v.3

### Installing and Executing

```
apt-get install git python

git clone https://github.com/r3nanp/portscanner/
```
 
```
cd portscanner
```

```
python portscanner.py
```
